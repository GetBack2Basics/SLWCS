# Using This Repository with AI Agents

This repository is structured to support AI-assisted job-application and profile-authoring workflows.

## Suggested AI Tasks

- Generate role-specific CV bullets
- Draft selection criteria responses
- Tailor a professional summary to a job description
- Produce concise project case studies from the projects folder
- Build interview talking points based on achievements and publications

## Prompt Templates

### 1) Tailored Professional Summary

"Using `docs/profile-overview.md`, `docs/key-achievements.md`, and `docs/skills-and-competencies.md`, draft a 120-word professional summary tailored to [ROLE/TITLE]."

### 2) Selection Criteria Response

"Using this repository, draft a STAR-format response to this criterion: [CRITERION]. Prioritize evidence from `projects/` and `experience/roles-and-responsibilities.md`."

### 3) Project-Based Achievement Bullets

"Create 8 resume bullets focused on measurable impact, using `projects/` and `publications/publications.md`. Group bullets under Technical, Leadership, and Stakeholder Engagement."


### 4) Interview Preparation

"Based on `docs/key-achievements.md` and `docs/timeline.md`, generate 10 likely interview questions and high-impact answer outlines."

## Best Practice

For formal submissions, always verify names, dates, and publication details against original source reports in `docs/sources.md`.
